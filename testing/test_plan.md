# План тестирования

## Проект: Веб-приложение с карточками товаров  
## Дата проведения: 10.11.2025  
## Цель: Проверить корректность работы основных функций приложения и удобство интерфейса для пользователей.

---

## Цели тестирования
## 1. Проверить корректность загрузки данных через `fetch` (работа с API).  
## 2. Убедиться, что карточки товаров отображаются корректно и быстро.  
## 3. Проверить отзывчивость интерфейса (адаптивность).  
## 4. Убедиться, что пользователь может удобно взаимодействовать с элементами интерфейса.  

---

## Среда и инструменты
- ## Операционные системы: Windows 10/11  
- ## Браузеры: Google Chrome, Mozilla Firefox, Microsoft Edge  
- ## Инструменты: Chrome DevTools, GitHub Issues, Google Forms  

---

## Типы тестирования
| Тип тестирования | Описание | Ответственный |
|------------------|-----------|----------------|
| Функциональное | Проверка работы `fetch`, отображения карточек, навигации | Разработчик |
| UI/UX тестирование | Проверка удобства интерфейса, визуальных элементов | Тестировщик |
| Адаптивность | Проверка корректности отображения на мобильных устройствах | Дизайнер |
| Производительность | Проверка скорости загрузки страниц | Разработчик |

---

## Критерии успешности
- Все основные функции работают без ошибок.  
- Загрузка данных из API происходит без задержек.  
- Интерфейс корректно отображается на разных устройствах.  
- Пользователи положительно оценивают удобство и понятность интерфейса.
