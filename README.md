# Attack on Titan Web Showcase

## Описание проекта

### Attack on Titan Web Showcase — это интерактивный веб-проект, посвящённый культовому аниме Attack on Titan.
### Он создан для поклонников серии, чтобы продемонстрировать визуальные сцены, видео, музыку и материалы, вдохновлённые вселенной аниме.
### Проект представляет собой веб-страницу с кастомным дизайном, видеофоном и встроенным музыкальным сопровождением из первого опенинга.

---
## Основные функции:
### Воспроизведение видео и аудио (Opening 1, клипы)

### Персональная страница Леви Аккермана (levi.html)

### Современный адаптивный дизайн с использованием SCSS

### Возможность стилизации и изменения тем

---

##  Установка и запуск

### 1. Склонируйте репозиторий
```
git clone https://github.com/username/attack-on-titan-web.git
```
### 2. Перейдите в каталог проекта
```
cd attack-on-titan-web
```
### 3. Запустите проект локально
```
start index.html
```

---

## Примеры использования

### При открытии index.html запустится главная страница с видеофоном и музыкой из первого опенинга.
```
<section class="two">
            <h1>Опенинг и мемы Атаки титанов</h1>
            <div class="twovideo">
                <video controls muted loop>
                    <source src="video.mp4">
                </video>
                <video controls muted loop>
                    <source src="IMG_2702.MP4">
                </video>
            </div>
            <audio controls>
                <source src="Attack on titan - Opening 1.mp3" type="audio/mp3">
            </audio>
</section>
```

### Страница levi.html содержит отдельный раздел, посвящённый персонажу Леви.
```
<h1>
  'Я хочу положить конец этому повторяющемуся кошмару прямо сейчас. Есть те, кто встанет у меня на пути.
  Но я отлично играю роль сумасшедшего, который убивает таких людей. Я должен быть готов устранить их.
  Потому, что я предпочитаю аду людей, убивающих друг друга, аду быть съеденным.'
</h1>
```

## Структура репозитория
```
attack-on-titan-web
│
├── img/                     # Каталог с изображениями проекта
├── Attack on titan - Opening 1.mp3  # Музыкальный файл (опенинг)
├── IMG_2702.MP4             # Видео-файл (основной клип или фон)
├── video.mp4                # Альтернативное видео
│
├── index.html               # Главная страница проекта
├── levi.html                # Страница персонажа Леви
│
├── style.css                # Основной файл стилей
├── xexe.scss                # Исходный SCSS-файл (стилизация)
├── xexe.css                 # Скомпилированный CSS-файл
├── xexe.css.map             # Source Map для SCSS
│
└── README.md                # Документация проекта
```

## Технические требования
